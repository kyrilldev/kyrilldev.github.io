Code HighLights and Explanations

The 'FadeIn' coroutine facilitates a smooth fade-in effect for UI elements using the 'CanvasGroup' component. Key features of this code include:

  • Time-based Fading: Utilizes a 'while' loop to incrementally adjust the alpha value of the 'CanvasGroup' from 1 (fully opaque) to 0 (fully transparent) over a specified 'fadeDuration'. This creates a gradual fade-in effect.
  • Interpolation for Smoothness: Implements 'Mathf.Lerp' to smoothly interpolate the alpha value based on the elapsed time, providing a visually pleasing transition.
  • Coroutine Yielding: Uses 'yield return null' to yield control back to the Unity engine each frame, ensuring that the coroutine doesn't block the main thread.
  • Interaction Disabling: After completing the fade-in, sets 'interactable' and 'blocksRaycasts' properties of the 'CanvasGroup' to false. This prevents interaction with the UI element while it is faded in.


    • The 'Choose' method handles the selection of choices within a game scenario. Key features of this code include:

  • Dynamic Choice Retrieval: Utilizes the Unity 'EventSystem' to retrieve the currently selected choice from the clicked button. The choice value is obtained from the 'Choice' component attached to the clicked game object.
  • Dynamic Element Spawning: Applies a switch statement on 'ChoiceIndex' to dynamically spawn an element based on the selected choice. The specific element to spawn is determined by the 'choice' value and the current 'ChoiceIndex'.
  • Array-Based Element Selection: Depending on the 'ChoiceIndex', selects the appropriate array ('choice1', 'choice2', etc.) and spawns the chosen element from that array.
  • Tracking Final Choices: Adds the selected choice to a 'finalCombo' list, providing a record of the player's choices.
  • Incremental Choice Index: Increments the 'ChoiceIndex' to move on to the next set of choices or elements.


    • The 'PopupAnimation' method orchestrates a dynamic animation for pop-up elements using position lerping. Key features of this code include:

  • Dynamic Popup Selection: Retrieves the specified popup element ('popupPlaceholder') from an array of popups ('popups') based on the provided 'index'.
  • Popup Position Toggle: Alternates between two popup positions ('popupPos') to create a visually engaging animation effect. The position toggle is achieved by incrementing or decrementing the 'popupPos' variable.
  • Position Lerping Coroutine: Initiates a coroutine ('PosLerp') to smoothly interpolate the position of the selected popup from its current position to the target position ('popupPositions[popupPos].position'). The lerping occurs over a duration of 1 second, contributing to a visually appealing and gradual popup movement.

    • The Assignment

    This project me and my group took part of a school wide gamejam. Day one we had to first find a group to work with of course so we the developers and the artists found each other in the cafeteria and started working on the idea for the game. The theme of the gamejam was 'reflection'. So after a bit of thinking we landed on a game where you could make/customize your own mirror by pressing buttons. And at the end of the game you get a self-reflect moment start by the game. But no matter what combination you press, you get called insane. We sadly didn't win any awards but it was a fun time none the less.
